    Struct SceneTree

    The scene tree, a hierarchy of all the game objects in the current scene.

    struct SceneTree;

    Constructors

    NameDescription
    this (name)

    Fields

    NameTypeDescription
    mGameObjects GameObject[]Dynamic array of game objects (for ease iterating through all game objects.)
    mRoot TreeNodeThe root node of the SceneTree. Usually contains a dummy game object.

    Methods

    NameDescription
    AddGameObject (go, parentId) Add a game object to the SceneTree as a child of the node whose game object matches parentId (or under the root node, if there is no such node.)
    GetGameObjectById (goId, n) Returns a reference to the GameObject matching the provided id in the SceneTree.
    GetGameObjectByType (type, n) Returns a reference to the first GameObject matching the provided type in the SceneTree. Most useful when there is only one object of a type.
    GetGameObjectNodeById (goId, n) Returns a reference to the TreeNode containing the GameObject matching the provided id.
    GetGameObjectNodeByType (type, n) Returns a reference to the TreeNode containing the first GameObject matching the provided type. Most useful when there is only one object of a type.
    GetGameObjects () Returns all game objects in the SceneTree.
    GetSceneTreeSize (n) Returns the number of game objects in the SceneTree.
    RemoveGameObject (goId, n) Remove the node from the SceneTree whose game object matches goId.
    Serialize ()
    TreeToString (n, depth) Returns a string representation of the SceneTree hierarchy, where more indented objects are children of less indented objects.
